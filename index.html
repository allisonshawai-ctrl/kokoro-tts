How to install

Start the Kokoro container

Runs locally, auto-starts on boot

docker run -d --name kokoro-tts --restart always -p 8199:8880 ghcr.io/remsky/kokoro-fastapi-cpu:latest

Run the patcher

Interactive — preview voices and pick your favorite

python scripts/patch_kokoro.py

Restart OpenClaw

Done. Hard refresh your browser.

openclaw gateway restart
